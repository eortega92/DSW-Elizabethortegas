ETIQUETAS BASICAS HTML

El lenguaje HTML (Hipertext Markup language) es el lenguaje con que se definen las paginas web. Permite describir el contenido de una página, incluyendo texto y otros elementos (imágenes, videos, documentos, etc.) HTML es la tecnología fundamental de la world wide web y ha ido evolucionando hasta nuestros días. HTML es un markup language, lo que significa que está escrito con códigos que puede leer una persona sin que sea necesario compilarlo primero. En otras palabras, el texto en una página web está «marcado» con estos códigos para dar instrucciones al navegador web sobre cómo mostrar el texto. Estas etiquetas de marcado son las propias etiquetas HTML.
